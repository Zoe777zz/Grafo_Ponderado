# Grafo_Ponderado

# Taller 10: Implementación del Algoritmo de Dijkstra
Este proyecto consiste en la implementación del Algoritmo de Dijkstra en el lenguaje de programación Java, aplicado a grafos ponderados. El objetivo principal es encontrar la ruta más corta entre un nodo origen y los demás nodos del grafo, comprendiendo los procesos de relajación de aristas y manejo de estructuras auxiliares.

# Instalación y Ejecución
1. Clonar el repositorio
Abre una terminal en tu computadora y ejecuta el siguiente comando:

Bash
git clone (https://github.com/Zoe777zz/Grafo_Ponderado.git)

2. Abrir el proyecto
Abre tu IDE (por ejemplo, VS Code).

Selecciona la opción "Open Folder" y busca la carpeta que acabas de clonar.

3. Ejecución
Localiza la clase principal (donde se encuentra el método main) y presiona el botón Run o presiona F5 en VS Code. El programa mostrará en consola:

Las distancias finales desde el nodo origen.

El camino mínimo reconstruido paso a paso.

# Estructura del Código
El desarrollo sigue el enfoque metodológico ABPr (Aprendizaje Basado en Proyectos)  y se organiza en:

Representación del grafo ponderado.

Lógica del algoritmo de Dijkstra (selección de nodo menor y relajación).

Módulo de salida de resultados en consola.
